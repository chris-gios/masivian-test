# Create Roulette

Allow to create a Roulette with closed status

**URL** : `/api/roulette`

**Method** : `POST`

**Data constraints** : NONE

**Header constraints** : NONE

**Data examples** : NO BODY

## Success Responses

**Condition** : NONE

**Code** : `201 Created`

**Content example** : Responses the roulette id and status.

```json
{
    "id": "5ef0844b07238162ebecdd46",
    "open": false
}
```
